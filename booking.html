  • ${fitur}
    • `; }); card.innerHTML = `

    ${paket.nama}

    ${paket.harga}

      ${fiturList}
    Pilih Paket `; container.appendChild(card); }); } // Jalankan saat halaman dimuat window.onload = tampilkanPaket;
